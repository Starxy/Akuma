网站内容

目前个人作为学习阶段还没有原创内容，发布的博客上都会标有原网站/作者的信息或者链接。若对您造成不良影响请及时联系我，会立刻删除并赔偿。

  • 主要作为个人学习笔记来记录知识点
    • 因学习阶段主要考虑个人情况，所记录的内容会随着个人的理解程度变更，使笔记更易本人使用
    • 主要记录需要记忆或容易混淆或者新学的部分
    • 对于个人认为极为简单和易于/已经掌握和不重要的部分不予记录
  • 也会记录一些出行游记和备忘
  • 菜谱。深夜放毒

Markdown 写作

Markdown 文件由 Visual Studio Code 编辑

  • markdwonlint 插件验证 Markdown 格式的规范性， 个人用户配置如下
    • 更改缩进为四个空格
    • 取消掉 h1 唯一的设定，原 h1 作为文章大标题并默认不展开 TOC (11 月 26 日添加)
  • Markdown + Math 预览 MathJax 公式
  • Markdown All in One 编辑 md 文件辅助功能
  • insert-spaces-between-half-fullwidths 提供在中文英文之间自动插入空格的功能，快捷键为Ctrl+Shift+J
